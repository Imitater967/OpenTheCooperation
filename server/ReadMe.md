## 架构

## 资金的流动
### 外部流入和流出

借： 银行存款
贷： 营业外收入-赞助者捐赠

借：管理费用-软件使用费     我们收取的费用
借：研发费用-某模块         研发人员的研发费用
借：管理费用-管理费  项目管理员（PullRequest,Issue Maintainer)的费用
贷：银行存款

### 内部流动

借：营业外收入-赞助者捐赠
贷：管理费用-某模块
贷：研发费用-某模块
贷：管理费用-管理费
